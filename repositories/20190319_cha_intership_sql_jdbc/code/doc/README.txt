# Desafio Estágio Full Stack 2019.1

## Sobre e tecnologias
O desafio estágio desenvolvedor 2019.1 é uma etapa do processo seletivo na empresa PariPassu. Neste desafio, foi solicitado um CRUD completo de uma base de dados em modelo relacional contendo múltiplas tabelas.

## As tecnologias envolvidas
Foi exigida a implementação utilizando Java JDBC utilizando o SGBD Postgres.

## Planejamento e Execução 
Foi disponibilizado dois dias para configuração do ambiente e 5h para estudo e implementação da solução. 

### O que foi desenvolvido com sucesso:
1. Foi possível configurar o ambiente com sucesso. 
2. Foi possível conectar no banco de dados. 
3. Foi possível desenvolver o CRUD completo das tabelas. 

### O que não foi de fato alcançado e por quê?
Ocorreram falhas para executar o CRUD com as tabelas de relacionamento. 
1. Pois foi a primeira vez foi programado um CRUD em Java com JDBC. 
2. Assim, as 5h disponibilizadas foram pouca.
3. Aluno de segunda-fase na época.
